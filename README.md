# Admin Dashboard

## This is the final project of the Intermediate HTML/CSS path where the goal is to create a dashboard  based on HTML/CSS intermediate fundamentals and apply the knowledge of grid/flexbox or combination of grid and flexbox in the design for seamless ui expreience. The following topics were implemented during the project: 

- HTML – using elements like `<main>`, `<section>`, `<form>` and more
- CSS Fundamentals – styling with custom fonts, spacing, colors, and transitions
- Flexbox – used to align navigation, structure the layout etc.
- Box Model – white space, full-width sections, padding/margin and `box-sizing`
- Grid - used to layout 2D element(columns/rows), structure and complex designs, offer precise control.

Live demo: https://deusvult09.github.io/admin-dashboard/
